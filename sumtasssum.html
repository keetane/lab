Practice written addition one cell at a time.

Sum Tass Sum helps children practice written addition with a guided grid, carry helpers, review, and local progress tracking for each profile.

Preparing for App Store

Learning flow

A focused practice loop for written addition.

Children choose an addition format, solve each cell in order, then review their own written work beside the correct layout.

Guided written addition

The board highlights the current answer or carry cell so learners can focus on one small step at a time.

Carry practice

Carry cells appear when needed, with hints and help tied to the current step.

Review after each set

Completed work is shown with mistakes and blanks marked for deliberate review.
